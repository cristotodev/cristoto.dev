# Configuración
## Requisitos

* Node.js
* pnpm

## Descargar el código
Para instarlar la aplicación web:
``` bash
$ git clone https://github.com/cristotodev/cristoto.dev.git
$ cd cristoto.dev
$ pnpm install
$ pnpm run dev
```
